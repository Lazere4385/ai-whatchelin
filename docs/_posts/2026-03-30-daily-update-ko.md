Claude Code가 해냈다. 오늘 90점에 도달하며 이 트래커 역사상 처음으로 일일 인기도에서 GitHub Copilot(89)을 추월했다. 이 이정표는 놀라운 12일간의 질주를 마무리한다: 3월 18일 83점에서 오늘 90점까지, Claude Code는 7점을 더하는 동안 Copilot은 제자리였다. 한편 OpenAI에 대한 문화적 반발은 깊어졌다 — #QuitGPT 운동이 250만 지지자를 넘었고, 국방부-Anthropic 분쟁 이후 ChatGPT 삭제가 295% 급증했다.

## Claude Code: 새로운 AI 개발 도구 2위

Claude Code의 90점 도달은 ChatGPT 다음으로 두 번째로 높은 AI 개발자 도구 순위를 확보했다. 견인력은 꾸준한 실행에 있다: 52일간 74회 제품 릴리스, ARR 25억 달러, Auto Mode, Computer Use 통합, 그리고 GitHub Copilot의 데이터 학습 논란에서 이익을 얻은 평판. ChatGPT는 여전히 96점으로 더 넓은 AI 어시스턴트 시장을 지배하지만, Claude Code는 이제 Microsoft가 지원하는 제품을 제치고 특히 개발자 도구 분야의 선두 자리를 차지했다.

## #QuitGPT: 250만 돌파 중

국방부-Anthropic 분쟁에 대한 개발자들의 불만으로 시작된 것이 측정 가능한 운동으로 성장했다. #QuitGPT 해시태그는 250만 명 이상의 지지자를 모았고, ChatGPT 삭제가 295% 급증했다 — 이 추세가 지속된다면 OpenAI의 참여 지표에 영향을 줄 수치다. 운동은 분산돼 있고 지속되지 않을 수 있지만, 한 주 만에 삭제 295% 급증은 단순한 잡음이 아닌 실제 신호다.

## Google: TurboQuant이 메모리 시장을 흔들다

Google Research가 H100 GPU에서 KV 캐시를 6배 줄이고 어텐션을 8배 빠르게 하는 벡터 압축 알고리즘 TurboQuant를 발표했다. 논문은 ICLR 2026에서 발표될 예정이다. 시장 파장은 즉각적이었다: SK하이닉스 6%, 삼성 5%, Micron 3.4% 주가 하락 — 투자자들은 TurboQuant를 고대역폭 메모리 수요 감소 가능성으로 읽었다. 반도체 애널리스트들을 긴장시키는 종류의 연구 논문이다.

## 개발자 도구 트렌딩

이번 주말 개발자 커뮤니티에서 여러 오픈소스 도구가 주목을 받았다: **Cog**(Claude Code 세션의 영구 메모리), **Expect**(AI 에이전트용 브라우저 테스트 프레임워크), **dev-browser**(CLI 기반 브라우저 제어), **n8n-mCP**(MCP를 활용한 워크플로우 자동화). Claude Code 생태계가 자연스럽게 확장되고 있다 — 플랫폼이 임계 규모에 도달하는 신호다.
