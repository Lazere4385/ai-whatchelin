이번 주 AI 도구 시장에 두 개의 폭탄이 터졌다. Cursor의 주력 모델인 Composer 2가 중국 AI 기업 Moonshot AI의 오픈소스 모델 Kimi K2.5를 기반으로 구축됐음에도 공개적으로 알리지 않았다는 사실이 드러났다 — 한 개발자가 API 응답에서 `kimi-k2p5`라는 모델 ID를 발견하기 전까지. 기업가치 500억 달러의 Cursor는 이 사실이 알려진 후에야 베이스 모델을 인정했다. 한편 OpenAI는 3월 24일 Sora 앱과 API를 완전히 종료하며 가장 주목받았던 AI 영상 생성기의 막을 내렸다.

## Cursor: Kimi K2.5 출처 미표기 스캔들

Cursor의 Composer 2는 중국 오픈소스 모델인 Moonshot AI의 Kimi K2.5를 기반으로 한 것으로 밝혀졌으며, 사용자나 개발자 커뮤니티에 이 사실이 공개되지 않았다. 이번 사건은 AI 제품 라벨링의 투명성에 대한 광범위한 논쟁을 촉발했다. 오픈소스 모델 위에 제품을 구축할 때 얼마나 출처를 밝혀야 하는가? Bloomberg에 따르면 Cursor의 기업가치는 최근 293억 달러에서 500억 달러 규모로 오른 협상이 진행 중이며(3월 12일), 이러한 규모에서의 침묵은 더욱 눈에 띈다. Cursor는 이후 베이스 모델을 인정했지만, 왜 처음부터 공개하지 않았는지는 해명하지 않았다.

## OpenAI: Sora 서비스 종료

OpenAI는 3월 24일 Sora 독립 앱과 API를 종료했다. 영상 생성 기능은 더 이상 단독 제품으로 제공되지 않으며, ChatGPT Plus와 Pro 구독 내에서만 이용 가능하다. 종료는 갑작스럽게 이뤄졌으며, API에 통합을 구축한 개발자들에게는 사전 예고가 거의 없었다.

## Gemini CLI: 무료 티어 축소

오늘부터 Gemini CLI 무료 티어는 Flash 모델로만 제한된다. 무료 사용자는 CLI에서 Gemini Pro 모델에 더 이상 접근할 수 없으며, Google AI Pro 또는 Ultra 구독이 필요하다. 이 변경 사항은 이달 초 GitHub 이슈 (#23049)와 커뮤니티 토론 (#22970)에서 먼저 알려졌다.

## GitHub Copilot: Gemini 3.1 Pro 추가

GitHub Copilot이 3월 23일 Gemini 3.1 Pro를 공개 미리보기로 추가했다. VS Code, JetBrains, Xcode, Eclipse에서 이용 가능하다. 이 추가로 Copilot 사용자는 IDE를 벗어나지 않고 Google의 가장 강력한 모델 중 하나를 쓸 수 있게 됐다.

## Windsurf: 가격 인상과 새 할당량 시스템

Windsurf의 가격 정책 개편이 시행됐다. 크레딧이 사라지고 일간·주간 할당량으로 대체됐다. Pro 티어 가격은 월 15달러에서 20달러로 인상됐다. 기존 15달러 구독자는 당분간 기존 가격이 유지된다.
