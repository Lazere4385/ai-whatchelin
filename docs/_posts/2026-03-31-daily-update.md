March ends with a pair of modest gainers and the broader landscape settling into a new configuration that would have looked unlikely three weeks ago. Cursor ticked up to 88 — recovering from the Kimi K2.5 attribution controversy that had pulled it to a tie with Claude Code earlier this month. Gemini CLI reached 88 on the back of its v0.35.2 stable release. Meanwhile, Antigravity continued its slow, steady slide to 55, now down 13 points from 68 in mid-March.

## Cursor: Regaining Ground

Cursor's gain to 88 is driven by two factors: the enterprise traction from its self-hosted cloud agents (launched March 25) and continued developer adoption of Composer 2, which despite the attribution controversy has been praised for its actual coding performance. Enterprise customers at Brex, Money Forward, and Notion represent the kind of design-win momentum that sustains growth past a PR setback. The Kimi K2.5 story has largely been absorbed by the market.

## Gemini CLI: v0.35.2 Stable

Gemini CLI's 63 — up one from yesterday — reflects the release of v0.35.2, the first stable version in several weeks. Stable releases matter to developers who had been on pre-release builds: they signal that Google considers the CLI production-ready. The free tier restriction to Flash models only (effective March 25) remains in place, but the stable release provides a cleaner baseline for teams evaluating the tool.

## Claude Code: Voice Mode (Retroactive Entry)

A previously missing entry was added to the Claude Code timeline today: Voice Mode, launched March 3, 2026. The /voice push-to-talk feature supports 20 languages and had been omitted from the official changelog until now. It's a minor housekeeping note, but Voice Mode represents a meaningful interface expansion for developers working in noisy environments or preferring hands-free interaction.

## GitHub Copilot: PR Bug Fix

GitHub Copilot shipped a fix for an issue where pull request summaries were displaying what appeared to be advertisement-style content — a bug that drew community ridicule. The fix is minor but the optics matter: for a tool already dealing with data training backlash, any additional trust erosion amplifies.

## Antigravity: 13 Points Down in Two Weeks

Antigravity's decline to 55 marks the end of a difficult March. The quota controversy — transitioning from predictable credit usage to daily and weekly caps — has visibly damaged developer confidence. Thirteen points lost in roughly two weeks is a pace that, if continued, will push Antigravity out of competitive contention within a month.
