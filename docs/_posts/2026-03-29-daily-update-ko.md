3월 28일 두 건의 주요 Claude 소식이 터졌고, 둘 다 같은 방향을 가리킨다: Anthropic이 문화적·법적 전선 모두에서 동시에 이기고 있다. Claude 앱이 App Store 무료 다운로드 차트에서 ChatGPT를 처음으로 추월했다 — AI 어시스턴트가 해당 지표에서 ChatGPT를 앞선 것은 이번이 처음이다. 몇 시간 앞서, 연방 판사는 회사가 자율 무기에 Claude 사용을 거부한 후 국방부가 Anthropic을 "공급망 위험"으로 지정하려는 시도를 차단했다. 두 헤드라인은 함께 Anthropic의 공개 포지셔닝에 있어 중요한 순간을 나타낸다.

## Claude Code: GitHub Copilot과 89점 동률

Claude Code가 오늘 89점에 도달해 ChatGPT 뒤 2위 자리에서 GitHub Copilot과 동률을 이뤘다. 3월 18일 83점에서 오늘 89점까지 단 11일 만의 상승은 이 추적 기간에서 가장 빠른 지속 상승이다. Claude AI도 1점 올라 85점을 기록했으며, App Store 소식에 힘입은 것으로 보인다.

## Anthropic 대 국방부

연방 판사가 Anthropic을 "공급망 위험"으로 지정하려는 국방부의 시도를 차단했다. 지정은 Anthropic이 자율 무기 응용을 위해 Claude를 제공하기를 거부한 후 추진됐다. 판결은 중요한 법적 순간이다: 회사가 국가 안보 위험으로 취급받지 않고도 AI의 특정 군사 용도를 거부할 수 있음을 확립했다. Claude의 특정 응용은 안전 원칙과 양립할 수 없다는 Anthropic의 입장이 사법적 검토를 통과했다.

## Claude, App Store 1위

Claude가 App Store 무료 차트에서 처음으로 ChatGPT를 앞질렀다. 타이밍이 주목된다: 국방부 판결과 진행 중인 #QuitGPT 운동(국방부-Anthropic 분쟁 이후 모멘텀을 얻었다)과 동시에 일어났다. App Store 급등이 지속 가능한지 아니면 뉴스 사이클의 일시적 현상인지가 핵심 질문이지만, 상징성은 실재한다.

## Gemini: Deep Think 출시

Gemini 3 Deep Think가 이제 Gemini Ultra 구독자에게 제공되며, 얼리 API 액세스도 열렸다. Deep Think는 확장 추론에 대한 Google의 답변 — 응답 전에 더 많은 컴퓨팅 시간을 숙고에 사용하는 모드다. API 접근은 Google이 개발자들이 이 기능을 기반으로 구축할 준비가 됐다는 신호다.

## Claude Mythos: 보도 지속

Opus 위의 새 티어인 유출된 "Capybara" — 역량의 "단계적 변화"로 묘사된 — 에 대한 보도가 주말 내내 이어졌다. 출시일은 아직 발표되지 않았다. 이야기는 개발자 커뮤니티 전반에서 계속 살아있다.
