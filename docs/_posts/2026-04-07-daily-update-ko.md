연초에는 거의 아무도 예상하지 못한 이정표를 Claude Code가 이번 주 넘어섰다. 92점으로 전체 AI 도구 순위에서 ChatGPT에 이어 2위 — GitHub Copilot을 처음으로 추월한 것이다. 이 움직임은 Claude Code의 일관된 제품 실행과 4월 24일 데이터 학습 마감을 앞두고 쌓여가는 Copilot의 신뢰 적자를 동시에 반영한다.

## Claude Code: 92점 신고점

90점에서 92점으로의 도약은 추적 기간 중 Claude Code의 최대 2점 단일 상승이다. 이번 주 초 출시된 v2.1.92 업데이트가 직접적으로 기여했다 — 60% 빨라진 Write 도구와 모델별 `/cost` 분석은 점수를 움직이는 종류의 개발자 중심 개선이다. 인터랙티브 Bedrock 설정 마법사는 이전까지 설정 과정을 복잡하게 여겼던 AWS 기업 사용자들에게 Claude Code의 문을 열어준다. Claude Code는 이제 우리가 추적하는 코딩 특화 도구 중 최고 점수를 보유했다.

## GitHub Copilot: 마감을 향한 하락

GitHub Copilot이 88점으로 내려앉았다 — 추적 시작 이래 최저점이다. 4월 24일 데이터 학습 옵트아웃 마감이 17일 앞으로 다가왔고, Reddit과 Hacker News의 개발자 여론은 개선되지 않고 있다. 핵심 불만 — 옵트아웃이 아니라 처음부터 옵트인이었어야 했다 — 이 얼리어답터 포럼에서 개발자 주류 담론으로 이동했다. Microsoft는 3월 말 불거진 PR 광고 버그에 대한 실질적인 해명을 아직 내놓지 않았다.

## Gemini CLI: 꾸준한 상승

Gemini CLI가 64점에 도달했다. 1점 상승은 v0.36.0 안정 릴리스의 지속적인 채택에 힘입은 것이다. 멀티 레지스트리 서브에이전트 아키텍처와 개선된 macOS/Windows 샌드박싱은 안정적인 로컬 실행이 필요한 팀들에게 호응을 얻고 있다. git 워크트리 지원은 이제 진지한 개발자 도구의 필수 기능이 됐다.

## Codex CLI: 레거시 모델 정리

Codex CLI v0.118.0이 레거시 gpt-5.1-codex 모델 지원을 제거했다. GPT-5.3 Codex Spark 통합을 위한 정지 작업으로, 점수는 72점을 유지했다. 실질적인 시험은 Spark가 리서치 미리보기에서 정식 출시로 전환될 때 찾아올 것이다.

## Sora: 카운트다운 시작

Sora 앱 종료(4월 26일)가 19일 앞으로 다가왔다. Kling, Runway, Vidu가 이탈 사용자를 계속 흡수하고 있다. ChatGPT 내 통합 영상 기능으로의 OpenAI 피벗이 주가 갈수록 더 뚜렷해지고 있다.

Claude Code와 Copilot의 궤적이 이제 뚜렷하게 갈라지고 있다. Claude Code가 이번 기간 2점을 얻는 동안 Copilot은 1점을 잃었다. 4월 24일 마감이 대규모 사용자 이탈을 촉발한다면 격차는 더욱 벌어질 수 있다. Cursor와 Copilot이 88점에서 동률 — Copilot과 Cursor의 경쟁이 주목해야 할 두 번째 스토리다.
