Cursor has officially overtaken GitHub Copilot for third place in our developer tool rankings, hitting a score of 89 versus Copilot's 87 — a lead driven by the momentum of Cursor 3's major relaunch and compounded by Copilot's ongoing data training controversy.

## Cursor: Version 3 Redefines the AI Editor

Cursor 3, released April 2, is not a minor update. The entire product has been rebuilt around the idea of parallel AI agents — the new Agents Window lets developers run multiple agents simultaneously across local machines, SSH remotes, and cloud environments. Design Mode takes a different approach: annotate any UI element in the browser and describe changes in plain language, no code required.

Also notable is Bugbot, which now integrates MCP support and has reached a self-reported 78–80% resolution rate — a figure that, if accurate, would make it among the most effective automated bug-fixing tools in the field.

## GitHub Copilot: Data Controversy Drags Scores Down

Copilot dropped a point to 87 as the data training opt-out backlash shows no sign of letting up. The April 7 release of CLI v1.0.21 added a new `copilot mcp` command for managing MCP servers, and the April 9 Autopilot preview introduced fully autonomous agent sessions — but neither release has been enough to turn sentiment around. Copilot's mobile push (Copilot cloud agent now on GitHub Mobile with signed commits) is a functional milestone, but community trust remains the core issue.

## Anthropic: Mythos Emerges from the Shadows

Anthropic formally launched a limited preview of Claude Mythos through Project Glasswing, a closed partner program involving 12 organizations including Amazon, Apple, Microsoft, and CrowdStrike. The model is reportedly deployed for defensive cybersecurity use cases, and at an estimated 10 trillion parameters it would be the largest publicly acknowledged AI system. Details remain sparse — Anthropic has not published a technical report.

## Gemini CLI: Version 0.37.0 Adds Browser Persistence

Gemini CLI v0.37.0 arrived April 8 with three notable additions: dynamic sandbox expansion, a "Chapters" feature that groups tool calls by topic for easier navigation, and a browser agent that now maintains persistent sessions across interactions. The update pushed Gemini CLI up one point to 65 — modest, but a sign of steady iteration.

## ChatGPT: CarPlay Integration Rolls Out

ChatGPT is expanding beyond the desktop with CarPlay support rolling out for iOS 26.4 and above. The integration enables hands-free conversations in supported vehicles, a move that positions OpenAI in the emerging in-car AI space. A new File Library feature also quietly landed, automatically saving files across sessions for easier reuse.
