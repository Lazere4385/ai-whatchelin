Two platform policy moves dominated the AI developer news on April 11: Anthropic blocked third-party subscription tools from accessing Claude, and GitHub paused new Copilot Pro trials citing abuse. Both decisions reflect a broader trend of AI companies pulling back permissive access as adoption scales.

## Anthropic: OpenClaw and Third-Party Tools Blocked

Anthropic revoked Claude subscription access for third-party clients including OpenClaw, a popular workaround that let users access Claude models at subscription rates through unofficial channels. Users of these tools now face cost increases of up to 50x — essentially the difference between a flat subscription and direct API pricing. The OpenClaw creator was also temporarily banned.

The move signals that Anthropic intends to enforce its terms of service more aggressively as Claude's commercial footprint grows. Developer reaction has been mixed: some see it as a legitimate business protection, others as a step backward for the open ecosystem around Claude.

## GitHub Copilot: Free Trial Pause Signals Conversion Struggles

GitHub paused new Copilot Pro trials effective April 10, citing abuse of the trial system. Existing trials and paid subscriptions remain unaffected. The pause is a notable admission — free trials are a standard growth tool, and halting them suggests conversion rates or abuse rates (or both) have crossed an internal threshold.

Copilot dropped another point to 86, its lowest in our tracking period. The combination of data training backlash, trial pause, and no major feature announcement is a difficult position to hold.

## Codex CLI: Autopilot Flags Land in v1.0.23

Codex CLI v1.0.23 shipped April 10 with three new startup flags: `--mode`, `--autopilot`, and `--plan`. These allow developers to launch directly into specific agent modes without navigating menus, a quality-of-life improvement that pushes the tool slightly upmarket. Codex CLI gained one point to 73 on the release.

## Claude Code: Team Onboarding and Computer Use in CLI

Claude Code v2.1.98, released April 9, added `/team-onboarding` workflows, automatic remote session setup, a 500K character cap for MCP contexts, and — most notably — computer use directly in the CLI. The ability to interact with a graphical interface from a terminal environment expands what Claude Code agents can autonomously accomplish.

## OpenAI: ChatGPT SuperApp and Gemini's 3D Visualizations

OpenAI's ChatGPT 5.5 SuperApp, launched April 6, consolidated ChatGPT, Codex, and the Atlas browser into a single desktop application. Meanwhile, Google Gemini (Pro) added a genuinely novel capability on April 10: interactive 3D models, physics simulations, and adjustable charts generated directly inside the chat interface. Neither company called attention to the other's move, but together they suggest the next competitive frontier is richness of in-context output.

Separately, OpenAI clarified the Sora shutdown timeline: the app closes April 26, the API follows on September 24.
