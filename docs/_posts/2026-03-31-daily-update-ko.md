3월이 소폭 상승한 두 도구와 함께 마무리됐고, 전체 판도는 3주 전에는 가능해 보이지 않았을 새로운 구성으로 자리를 잡았다. Cursor가 88점으로 올라섰다 — 이달 초 Claude Code와 동률을 만들었던 Kimi K2.5 출처 논란으로부터 회복하며. Gemini CLI는 v0.35.2 안정 릴리스를 타고 63점에 도달했다. 한편 Antigravity는 3월 중순 68점 대비 이제 13점 빠진 55점으로 완만하고 꾸준한 하락세를 이어갔다.

## Cursor: 잃었던 자리 되찾기

Cursor의 88점 상승은 두 가지 요인에 힘입었다: 3월 25일 출시한 자체 호스팅 클라우드 에이전트의 기업 견인력과 Composer 2에 대한 지속적인 개발자 채택. 출처 논란에도 불구하고 Composer 2는 실제 코딩 성능으로 호평을 받았다. Brex, Money Forward, Notion의 기업 고객은 홍보 위기를 넘긴 성장 모멘텀을 지속시키는 종류의 레퍼런스 승리다. Kimi K2.5 이야기는 시장에 의해 대부분 흡수됐다.

## Gemini CLI: v0.35.2 안정 릴리스

Gemini CLI의 63점 — 어제보다 1점 상승 — 은 몇 주 만의 첫 안정 버전인 v0.35.2 릴리스를 반영한다. 안정 릴리스는 프리릴리스 빌드를 사용해온 개발자들에게 중요하다: Google이 CLI를 프로덕션 준비 완료로 간주한다는 신호이기 때문이다. 3월 25일부터 시행된 무료 티어의 Flash 모델 전용 제한은 유지되지만, 안정 릴리스는 도구를 평가하는 팀에게 더 명확한 기준선을 제공한다.

## Claude Code: Voice Mode (사후 추가)

오늘 Claude Code 타임라인에 이전에 누락됐던 항목이 추가됐다: 2026년 3월 3일 출시된 Voice Mode. /voice 푸시 투 토크 기능은 20개 언어를 지원하며 공식 changelog에서 그동안 빠져있었다. 작은 정리 노트이지만, Voice Mode는 소음 환경에서 작업하거나 핸즈프리 상호작용을 선호하는 개발자에게 의미 있는 인터페이스 확장을 나타낸다.

## GitHub Copilot: PR 버그 수정

GitHub Copilot이 풀 리퀘스트 요약이 광고처럼 보이는 콘텐츠를 표시하는 문제를 수정했다 — 커뮤니티의 조롱을 샀던 버그다. 수정은 사소하지만 이미지가 중요하다: 데이터 학습 반발을 이미 다루고 있는 도구에게 추가적인 신뢰 훼손은 상황을 악화시킨다.

## Antigravity: 2주 만에 13점 하락

Antigravity의 55점 하락은 힘든 3월의 마무리를 알린다. 할당량 논란 — 예측 가능한 크레딧 사용에서 일간·주간 상한으로의 전환 — 이 개발자 신뢰에 가시적인 손상을 입혔다. 대략 2주 만에 13점 하락은, 이 속도가 계속된다면 한 달 내에 Antigravity를 경쟁 대열에서 밀어낼 페이스다.
