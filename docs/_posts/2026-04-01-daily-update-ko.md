Anthropic이 Claude Code와 함께 첫 번째 대형 인프라 위기에 직면했다. 회사는 사용자들이 "예상보다 훨씬 빠르게" 사용 한도에 도달하고 있음을 공식 인정했다 — 피크 시간대 스로틀링, 프롬프트 캐싱 버그, 2배 오프피크 프로모션 종료가 복합적으로 작용한 결과다. 해결 일정은 아직 없다. 한편 3월 31일 Copilot 광고 논란은 가라앉지 않고 있으며, 4월 24일 데이터 학습 옵트아웃 마감이 압박을 가중시키고 있다.

## Claude Code: 할당량 위기

Claude Code 할당량 위기는 현재 AI 개발자 커뮤니티의 최대 화제다. Reddit과 Hacker News에는 세션 도중 한도 벽에 부딪혔다는 개발자들의 게시물이 줄을 잇고 있으며, Anthropic의 유일한 안내는 기다리거나 사용량을 줄이라는 것뿐이다. 캐싱 버그가 특히 뼈아프다 — 토큰 소비가 적어야 할 작업에서 전체 할당량을 소진하고 있기 때문이다. 이 도구가 우리 추적 점수에서 GitHub Copilot을 막 추월한 시점이라 신뢰도 타격이 더 크다.

## GitHub Copilot: 광고 논란 후폭풍 지속

3월 31일 PR 광고 버그 — Copilot이 개발자의 풀 리퀘스트에 홍보 콘텐츠를 잠깐 삽입한 사건 — 가 r/programming 최상위 화제로 남아 있다. Microsoft는 아직 상세한 사후 분석을 내놓지 않았다. 4월 24일 데이터 학습 옵트아웃 마감이 겹치며 여론은 악화일로다. 개발자들은 Copilot의 기본 옵트인 방식이 우선순위 변화를 반영하는 건 아닌지 의구심을 품고 있다.

## Microsoft Copilot: Critique + Council

긍정적인 소식도 있다. Microsoft의 새 Copilot Critique and Council 기능이 3월 30일 출시됐다. GPT가 초안을 작성하면 Claude가 정확성을 검토하는 멀티모델 구조로, DRACO 벤치마크에서 +13.88% 향상을 기록했다. Microsoft가 품질 보증을 위해 Anthropic의 모델에 명시적으로 기대는 것은 이례적인 행보다.

## Antigravity: 꾸준한 하락세

Antigravity가 1점 더 하락해 54점을 기록했다. 미해결 할당량 논란과 Cognition 인수 불확실성이 맞물린 하향세다. 팀이 우려를 해소하는 속도보다 커뮤니티 신뢰가 더 빠르게 무너지고 있는 모양새다.
